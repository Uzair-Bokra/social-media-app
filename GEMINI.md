# complete social media posting system

## Project Overview
Professional social media system applicationn with, Post creation, Like system, Delete post, Search,
Sorting/Filtering, Clean UI, Mobile-friendly layout - similar to FaceBook applications.

## Core Features
- Signup / Login Flow (Front-End Only)
- Create Post Section
- Posts Feed
- Like System
- Delete Post
- Search Posts
- Sort & Filter Controls

## Tech Stack (JavaScript Version)

- Language: JavaScript (Browser only)
- UI: HTML and CSS
- Backend: None (all logic runs in the browser)
- Storage: LocalStorage for signup, login and other data
