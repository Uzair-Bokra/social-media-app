## Goal
Build the All feature - Create Post Section, Posts Feed, Like System, Delete Post, Search Posts, Sort & Filter Controls.

## Learning Focus
- creating post section
- liking post
- deleting post
- searching post
- sorting and filtering

## Fintech Concepts
- **Post Section**: If a man is login so he can post 
- **Posts feed**: if posted so all the post will show down like a post feed
- **Search posts**: we can search the post
- **sort and filter**: sorting and filtering the post


## Features to Build

### 1. Create Post Section
Users should be able to:
- Write a post (text only and image Url)
- Add emojis like like we can add multiple emojis in linkedin
- Click Post button to publish
- Show new post on top of feed (latest-first)

### 2. Posts Feed
Each post must display:
- Posted text
- Time/Date
- Like button
- Like counter
- Delete button

### 3.  Like System
- Clicking the heart should toggle Liked / Unliked state
- Counter should update (1 Like → 0 → 1)

### 4. Delete Post
- Remove post from DOM
- Optional: Ask for confirmation

### 5. Search Posts
- Search bar at the top
- Filter posts by text content

### 6. Sort & Filter Controls
Add 2–3 filters like:
- Latest First
- Oldest First
- Most Liked

### 7. Edit Post
- Click a button → open edit popup → update post.



### 7 UI
- make the best of the best UI
